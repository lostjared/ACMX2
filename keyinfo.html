ACMX2 Keyboard Controls

Real-time GLSL Shader Processing & Video Effects

Shader Management

↑ Up Arrow

Previous Shader

Cycles backward through your shader library. Instantly loads the previous GLSL shader in the collection, allowing you to quickly compare different visual effects.

↓ Down Arrow

Next Shader

Cycles forward through your shader library. Move to the next shader in your collection to preview different effects in real-time.

Space

Enable/Disable Processing

Toggles shader processing on/off. When disabled, displays the raw input video/camera feed without any shader effects applied. Useful for performance comparison or viewing the original source.

Playback Controls

L

Freeze Video Video/Image Mode

Freezes the current frame while continuing shader processing. The video playback stops, but shaders continue to animate on the frozen frame. Perfect for examining specific moments or creating static effect compositions.

P

Pause Video Video/Image Mode

Pauses both video playback and shader processing. Everything stops - the video frame and shader animations. Resume by pressing P again. Useful for inspecting a particular frame without shader animation.

Time & Animation Control

T

Enable/Disable Time

Toggles the shader time uniform. When disabled, the time value stops incrementing, freezing all time-based shader animations. Shaders that depend on iTime will stop animating.

I / O

Step Time Forward/Backward

I - Increment time by one frame step
O - Decrement time by one frame step
Only works when time is disabled (press T first). Allows frame-by-frame control of time-based animations.

Q

Reactive Time Audio Mode

Toggles audio-reactive time mode. When enabled, the shader time progression responds to audio input amplitude, creating music-synchronized visual effects. Requires AUDIO_ENABLED build flag.

Camera & View Control

W / A / S / D

3D Camera Movement 3D Mode

W - Look up (pitch up)
S - Look down (pitch down)
A - Rotate left (yaw left)
D - Rotate right (yaw right)
Navigate the 3D space when rendering shaders on 3D geometry. Provides first-person camera controls.

V

Auto-Rotation 3D Mode

Toggles automatic view rotation in 3D mode. When enabled, the camera automatically rotates around the 3D object, providing a continuous 360° view without manual input.

Capture & Display

Z

Take Snapshot

Captures the current frame as a PNG image. Saves to the configured screenshot directory with timestamp and resolution in the filename (e.g., ACMX2.Snapshot-2025.11.14-15.30.45-1920x1080-0.png).

F

Toggle Fullscreen

Switches between windowed and fullscreen mode. Provides an immersive viewing experience and better performance by removing window decorations and OS overhead.

Note: Some controls are mode-specific:
  • Video/Image Mode: L (freeze) and P (pause) only work when playing video files or displaying images
  • Audio Mode: Q (reactive time) requires the program to be compiled with AUDIO_ENABLED
  • 3D Mode: W/A/S/D camera controls and V (auto-rotation) only function when 3D mode is enabled
